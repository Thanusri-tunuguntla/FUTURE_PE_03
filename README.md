# FUTURE_PE_03
# AI SEO Blog & Content Cluster Generator

This project was created as part of the Prompt Engineering Internship at Future Interns.

The goal is to design a prompt system that generates SEO optimized blogs and content clusters for business websites.

Business Niche:
Digital Marketing Agency in Bangalore

Project Includes:
- SEO pillar blog
- Supporting blog articles
- Keyword research
- Structured prompt templates

Tools Used:
ChatGPT
Google Search
Google Trends

SEO Strategy:
Content cluster model where one pillar article is supported by multiple related blogs.

This approach improves:
- search engine ranking
- topical authority
- internal linking
- organic traffic
